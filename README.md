# HTML Series 📘

## 📌 Description
HTML series is a collection of lessons and projects covering HTML basics to advanced topics like tags, forms, tables, and layouts for web development learning.

## 🛠️ Topics Covered
- HTML Basics  
- Tags & Elements  
- Forms & Inputs  
- Tables  
- Semantic HTML  
- Page Layouts  

## 📂 Structure

/html-series
│── lesson1.html
│── lesson2.html
│── forms.html
│── tables.html


## 🚀 How to Use
Open any `.html` file in your browser to view the lessons.

## 🎯 Purpose
To build strong foundational knowledge of HTML for web development.
